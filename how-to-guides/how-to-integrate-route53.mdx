## Overview

This guide explains how to integrate Amazon Route53 with Cloudsania, enabling you to manage your DNS records, hosted zones, and domain registrations directly from your Cloudsania dashboard.

## Prerequisites

- A Cloudsania account
- AWS account with Route53 access
- IAM credentials with appropriate permissions
- Basic understanding of DNS concepts

## Integration Steps

### 1. Configure AWS IAM Permissions

Integrating  Route53 with Cloudsania allows for seamless **domain management, DNS automation, and infrastructure scalability**. With this integration, you can manage DNS records and streamline cloud-based network services directly from the Cloudsania platform

<Note>
  If you have already set up an **AWS Cloud Connector**, the integration is automatically enabled. If not, follow the [**AWS**](https://developer.cloudsania.com/cloudServiceProviders/aws)[ **Cloud Connector Setup Guide**](https://developer.cloudsania.com/cloudServiceProviders/aws) .
</Note>
