## Overview

This guide walks you through setting up a DigitalOcean connector in Cloudsania, enabling you to manage your DigitalOcean resources efficiently.

## Prerequisites

- A Cloudsania account
- DigitalOcean account
- API token with appropriate permissions
- Basic understanding of DigitalOcean services

## Setup Steps

### 1. Generate DigitalOcean API Token

1. Log in to DigitalOcean
   - Go to API section
   - Navigate to Tokens/Keys
   - Click "Generate New Token"

2. Configure Token
   - Set token name
   - Select appropriate scopes:
     - Read access
     - Write access
   - Set token expiration (optional)

3. Save Token
   - Copy the generated token
   - Store it securely
   - Note: Token won't be shown again

### 2. Configure Cloudsania Connector

1. Access Cloud Connectors
   - Log in to Cloudsania
   - Navigate to **Cloud Connectors**
   - Select **DigitalOcean**

2. Add DigitalOcean Configuration
   - Click "Add DigitalOcean Connector"
   - Enter connector name
   - Paste API token
   - Configure default region

3. Verify Connection
   - Test API access
   - Validate permissions
   - Check resource access
