## Overview

This guide explains how to deploy a Kubernetes stack (Konstack) in Cloudsania, enabling you to run containerized applications efficiently.

## Prerequisites

- A Cloudsania account
- An environment set up
- Cloud provider credentials configured
- Basic understanding of Kubernetes concepts

## Steps by Step

### 1. Access Konstack Section

1. Log in to your Cloudsania dashboard
2. Navigate to **Environments**
3. Select your target environment
4. Go to the **Konstacks** section

## &#x20;**How To Deploy a Konstack**

To deploy a new **Konstack Instance**, follow these steps:

### **Step 1: Select Your P**referred Compute Konstack

1. Navigate to the Projects tab in the Cloudsania dashboard.

2. Locate the Project and environment where you want to deploy the Konstack for.

3. Click on the \`View Environmen&#x74;**'** option to access the environment's details.

4. Under the 'Konstack&#x73;**'** tab, you will find all the available compute stacks.

5. Click on the preferred Compute Konstack to start the configuration process.

### **Step 2: Configure the Konstack Instance**

* Enter an 'Instance Name' that describes your deployment (e.g, clousania-dev-server)

* Choose an 'Instance Type' (e.g., t2.micro, t2.nano, t2.small, etc.).

* Set the Minimum Active Instances (ensures availability).

* Set the Maximum Active Instances (defines the scaling limit).

### **Step 3: Complete the** Konstack **Setup**

* Click 'Setup Konstac&#x6B;**'** to deploy the instance.

* Upon successful deployment, the new instance will now appear under 'My Konstack&#x73;**'** with details on status, date created, and assigned cloud region.

## **Managing Konstacks**

### **Viewing an Existing Konstack**

* Under the **Konstack tab**, select **My Konstacks** to see the list of all configured Konstacks.

* Click on **View** to access the instance details.
