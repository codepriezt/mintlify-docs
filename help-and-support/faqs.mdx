## General Questions

<AccordionGroup>
  <Accordion title="What is Cloudsania?">
    Cloudsania is a cloud deployment platform that simplifies infrastructure management across multiple cloud providers. It enables you to deploy, manage, and monitor your applications seamlessly across different cloud environments.
  </Accordion>

  <Accordion title="Which cloud providers do you support?">
    We currently support:
    - Amazon Web Services (AWS)
    - VPS  (Virtual Private Servers) like NameCheap
    - DigitalOcean
    
    We're continuously adding support for more providers.
  </Accordion>

  <Accordion title="How does pricing work?">
    Our pricing is based on your usage and selected plan. We offer:
    - Free tier for small projects
    - Professional tier for growing teams
    - Enterprise tier for large organizations
    
    Visit our [pricing page](https://cloudsania.com/pricing) for detailed information.
  </Accordion>
</AccordionGroup>

## Technical Questions

<AccordionGroup>
  <Accordion title="How do I connect my cloud provider?">
    You can connect your cloud provider through our Cloud Connectors:
    1. Navigate to Cloud Connectors in your dashboard
    2. Select your provider
    3. Follow the authentication steps
    4. Verify the connection
    
    See our [Cloud Connector guide](/cloud-connectors/cloud-connectors) for detailed instructions.
  </Accordion>

  <Accordion title="What is a Konstack?">
    A Konstack is a pre-configured infrastructure template that defines your application's cloud resources and configurations. It helps you maintain consistency across deployments and environments.
    
    Learn more in our [Konstack documentation](/environments/konstacks).
  </Accordion>

  <Accordion title="How do I set up CI/CD pipelines?">
    Setting up CI/CD pipelines involves:
    1. Connecting your repository
    2. Configuring build settings
    3. Defining deployment environments
    4. Setting up triggers
    
    Check our [CI/CD guide](/how-to-guides/how-to-setup-a-cicd-pipeline) for step-by-step instructions.
  </Accordion>
</AccordionGroup>

## Security & Compliance

<AccordionGroup>
  <Accordion title="How secure is Cloudsania?">
    We implement industry-standard security measures:
    - End-to-end encryption
    - Role-based access control
    - Regular security audits
    - Multi-factor authentication
  </Accordion>

  <Accordion title="Do you store my cloud credentials?">
    We never store your cloud provider credentials. We use secure token-based authentication and encrypt all sensitive data in transit and at rest.
  </Accordion>
</AccordionGroup>

## Account Management

<AccordionGroup>
  <Accordion title="How do I add team members?">
    To add team members:
    1. Go to Account Settings
    2. Select Team Management
    3. Click "Add Member"
    4. Enter their email and role
    
    See [Team Management](/account-settings/team-management) for more details.
  </Accordion>

  
</AccordionGroup>
