Add-ons in Cloudsania provide additional functionalities that enhance the capabilities of your environment. These include database storage, caching solutions, and other essential services required to run applications efficiently. Users can configure and deploy add-ons directly within their environments.

## Accessing the Add-ons Page

To access the **Add-ons** page:

1. Navigate to the **Projects** section from the sidebar.

2. Select a project with an existing environment.

3. Click on the **Environment Name** to open its details.

4. Click on the **Add-ons** tab to view available add-ons.

## Available Add-ons

Currently, Cloudsania provides storage add-ons for relational databases:

* **PostgreSQL** - A powerful open-source relational database management system.

* **MySQL** - A widely-used relational database for structured data management.

## Setting Up an Add-on

To set up a new database add-on, follow these steps:

### Step 1: Choose an Add-on

1. On the **Add-ons** page, navigate to the **Featured Add-ons** section.

2. Select either **PostgreSQL** or **MySQL**.

### Step 2: Configure Add-on Settings

1. Provide an **Add-on Name** to identify the instance.

2. Enter the **Database Name** (e.g., `postgres-db` for PostgreSQL or `mysql-db` for MySQL).

3. Define the **Database Username** and **Password** for authentication.

4. Click **Setup Add-ons** to deploy the database.

### Step 3: Verify Deployment

1. Once setup is complete, the add-on will be listed under **My Add-ons**.

2. You can view and manage the instance from the **Add-ons** page.
