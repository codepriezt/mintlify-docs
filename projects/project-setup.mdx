## <Icon icon="diagram-project" color="orange" iconType="sharp-regular" size={35} />  What are Projects in Cloudsania?

<Note>
  Projects in Cloudsania represent  a tech product, website or any software application  designed to solve specific use cases. An Example of a project is [***Cloudsania***](https://www.cloudsania.com/) ***,** a solution that is designed for users to easily configure, deploy and monitor all of their cloud operations across various cloud providers within a central view.*
</Note>

## **Project Modes**

When setting up a project, Cloudsania offers three different setup modes to cater to various user preferences and expertise levels:

<Tabs>
  <Tab title="Easy Mode (App Services)">
    This mode is designed for users who want a hassle-free experience without requiring in-depth cloud knowledge. It provides pre-configured [Konstacks](https://developer.cloudsania.com/environments/konstacks) to simplify deployment, making it the best choice for beginners or those looking for a quick setup.
  </Tab>

  <Tab title="Developer Mode">
    Developer Mode is suitable for tech-savvy users who prefer manual configurations. It allows users to define their own cloud environments, set up DNS configurations, configure Konstacks (pre-defined stacks), and integrate CI/CD pipelines.

    Users opting for this mode should have basic knowledge of cloud environments and configurations
  </Tab>

  <Tab title="Professional Mode">
    Professional Mode is tailored for organizations requiring expert guidance. With this option, Cloudsaniaâ€™s internal team of skilled solution architects will set up the project based on its unique requirements, ensuring optimal architecture and adherence to best practices.
  </Tab>
</Tabs>
